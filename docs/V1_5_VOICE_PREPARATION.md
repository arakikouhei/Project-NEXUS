# Project NEXUS v1.5 音声機能準備

## 目的

v1.5では、将来の音声操作に向けた準備を行う。

ただし、この段階では実際の音声入力・音声出力はまだ実装しない。

## 将来必要になるもの

- マイク入力
- 音声認識
- テキスト化
- NEXUSコマンドへの変換
- 音声応答
- 読み上げ
- 起動ワード
- 誤作動防止
- ローカル処理優先
- 有料API利用時の明示確認

## 安全方針

音声操作で危険な操作を直接実行しない。

音声だけで実行しないもの：

- ファイル削除
- Git commit
- Git push
- 任意シェル実行
- 有料API呼び出し
- 外部送信

## v1.5完了条件

- 音声機能の前提が整理されている
- 危険操作の防止方針がある
- ローカル処理優先の方針がある
