# Paper Intake v2

Adds paper organization tools for saved arXiv entries.

## Commands

- 論文要点整理: papers-xxxxxxxx
- 論文キーワード抽出: papers-xxxxxxxx
- 論文3行まとめ: papers-xxxxxxxx
- 論文安全評価: papers-xxxxxxxx

## Scope

This version uses metadata and Abstract stored in Knowledge Core.
It does not read or redistribute full PDFs.
