# Memory Index v1

Adds memory overview commands for Project NEXUS v0.6.

## Commands

- 記憶インデックス
- 記憶カテゴリ一覧
- 記憶重要項目
- 記憶ファイル一覧

## Purpose

Memory Index v1 shows the current state of NEXUS memory-related data:

- Project Memory
- Knowledge Core
- Source Registry
- World Update logs
- Release snapshots
- Important docs
- Imported notes
- Research notes

## Safety

- Read-only
- Does not delete memory
- Does not rewrite knowledge
- Used for overview and navigation
