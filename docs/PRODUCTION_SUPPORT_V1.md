# Production Support v1

Production Support v1 adds creative production memo commands for Project NEXUS v0.7.

## Commands

- 制作メモヘルプ
- 制作メモ追加: text
- 制作メモ一覧
- 制作メモ検索: query
- 制作メモ詳細: memo-id
- 3DCG作業確認
- Maya作業メモ

## Purpose

- Store creative production notes
- Support 3DCG / Maya workflows
- Prepare for future dashboard production panels

## Data

Production memos are stored in:

- data/production/production_notes.json

## Safety

- Does not edit source files
- Does not delete notes
- Keeps production notes separate from system memory and work notes
