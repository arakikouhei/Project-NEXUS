# Project NEXUS v2.0 安全方針

## 目的

球体AIデバイス化に向けた安全方針を定義する。

## 基本原則

NEXUSはユーザー補助AIであり、ユーザーの許可なく危険な操作をしない。

## 禁止する自律操作

- ファイル削除
- ファイル上書き
- Git commit
- Git push
- 任意シェル実行
- 有料API利用
- 外部送信
- 個人情報送信
- 常時録音
- 常時カメラ使用

## 確認が必要な操作

- 外部通信
- API利用
- ファイル編集
- 設定変更
- 長時間処理
- データ送信
- ハードウェア制御

## 音声機能の安全

音声だけで危険操作を完了しない。

危険操作は必ず画面またはターミナルで確認する。

## ハードウェア安全

- バッテリー異常時は停止
- 温度異常時は停止
- 充電異常時は停止
- センサー異常時は警告
- 冷却異常時は警告

## v2.0での結論

安全方針を先に固定してから、音声・通信・ハードウェア機能を追加する。
