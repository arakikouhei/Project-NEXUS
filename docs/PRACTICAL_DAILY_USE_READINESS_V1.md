# 日常利用準備チェック v1

## 目的

Project NEXUS が毎日使える補助AIとして使える状態か確認する。

## 必要なこと

- 現在地を確認できる
- 次にやることを確認できる
- システムの健康状態を確認できる
- 記憶を確認できる
- 重要ファイルを確認できる
- 制作メモを確認できる
- ダッシュボードを起動できる
- 安全にGit保存できる

## 代表コマンド

- NEXUS現在地
- NEXUS開発状況
- システム健康診断
- おすすめ次操作
- NEXUS記憶
- 重要ファイル一覧
- ファイルインデックス
- NEXUSダッシュボード起動方法

## ダッシュボード起動

python3 -m nexus.dashboard.server

URL：

http://127.0.0.1:8765

終了：

Ctrl + C

## 安全ルール

ダッシュボードから以下は行わない。

- 任意のシェル実行
- ファイル削除
- ファイル編集
- Git commit
- Git push
- 有料API呼び出し
