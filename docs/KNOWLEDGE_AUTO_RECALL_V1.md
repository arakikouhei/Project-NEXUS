# Knowledge Auto Recall v1

Adds conservative automatic recall when Knowledge Auto Recall is ON.

## Examples

- MayaのUVってどう使うの？
- PointDiTって何？
- diffusionって3DCGに関係ある？

## Scope

Only works when 知識自動参照ON is enabled.
Does not take over existing commands, casual chat, translation, or language practice.
