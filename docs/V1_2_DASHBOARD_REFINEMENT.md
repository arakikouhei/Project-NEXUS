# Project NEXUS v1.2 ダッシュボード改善

## 目的

v1.2では、ダッシュボードをより日常利用しやすくする。

v1.0時点では、ローカルダッシュボードの土台と安全方針がある。
v1.2では、表示内容と操作導線を整理する。

## 改善対象

- 現在地表示
- システム状態表示
- 次作業表示
- 重要ファイル表示
- 記憶状態表示
- 制作メモ表示
- 起動方法の案内
- 終了方法の案内

## 安全方針

ダッシュボードには危険な操作を入れない。

入れないもの：

- 任意のシェル実行
- ファイル削除
- ファイル編集
- Git commit
- Git push
- 有料API呼び出し

## v1.2完了条件

- 表示中心の安全なUI方針がある
- ローカル運用の前提が明確
- 将来のパネル拡張方針がある
