# Vision Foundation v1

This is the first local vision layer for Project NEXUS.

## Commands

- 画像ヘルプ
- 画像安全確認: /path/to/image.png
- 画像分析: /path/to/image.png

## Current Capabilities

- file safety check
- image format
- width and height
- brightness
- contrast
- saturation
- dominant colors
- simple visual hints

## Not Yet

- object recognition
- exact plant identification
- exact mineral identification
- live camera analysis
- attack/impact detection

These are planned for later vision model stages.
