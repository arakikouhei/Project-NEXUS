# Project NEXUS v2.0 球体AIデバイス計画

## 目的

v2.0では、Project NEXUSを将来の球体AIデバイスへ進めるための本格設計段階に入る。

ここでのv2.0完成は、実物ハードウェア完成ではない。

v2.0完成の意味は、次の通り。

- 球体AIデバイス化の設計方針がある
- ソフトウェアとハードウェアの接続方針がある
- 安全方針がある
- 将来実装の順番がある
- 現在のMac上NEXUSから移行する道筋がある

## 想定デバイス

形状：

- 球体
- 底面は自立用に少し平ら
- ハンドボール大を想定

外部要素：

- 上面小型ディスプレイ
- USB / Type-C
- SDスロット
- 音声入力
- 音声応答
- ワイヤレス接続

内部要素：

- 中央CPUコア
- 同心円PCBスタック
- 底部バッテリー
- 中央ジャイロ / 慣性センサー
- 上部アンテナ
- 冷却機構
- 将来的な液冷検討

## v2.0で決めること

1. 球体AIデバイスの役割
2. ローカルAI優先方針
3. 音声機能の安全設計
4. ディスプレイ表示内容
5. センサーと状態監視
6. ネットワーク安全方針
7. 将来のハードウェア構成
8. Mac上のNEXUSとの関係
9. 有料APIを使わない基本方針
10. 物理デバイス化までのロードマップ

## v2.0でまだ作らないもの

- 実物の球体筐体
- 実物の基板
- 実物の液冷機構
- 実物のバッテリー制御
- 実物の音声入出力
- 実物のカメラ
- 常時起動AI
- 自律的な外部通信
- 有料API自動利用

## 安全方針

球体AIデバイスは、ユーザーの補助を目的にする。

禁止方針：

- ユーザー確認なしで外部送信しない
- ユーザー確認なしで有料APIを使わない
- ユーザー確認なしでファイルを削除しない
- ユーザー確認なしでGit操作しない
- ユーザー確認なしでシェル命令を実行しない
