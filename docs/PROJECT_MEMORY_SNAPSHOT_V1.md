# Project Memory Snapshot v1

Adds snapshot commands for Project NEXUS Project Memory.

## Commands

- NEXUS記憶スナップショット
- NEXUS記憶履歴
- NEXUS記憶復元候補

## Purpose

Save dated snapshots of:

- data/project/project_memory.json

## Safety

- Does not delete memory
- Does not automatically restore
- Restore candidates are display-only in v1
