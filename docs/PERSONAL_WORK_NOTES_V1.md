# Personal Work Notes v1

Adds personal work note commands for Project NEXUS v0.6.

## Commands

- 作業メモヘルプ
- 作業メモ追加: text
- 作業メモ一覧
- 作業メモ検索: query
- 作業メモ詳細: memo-id

## Data

- data/work_notes/work_notes.json

## Safety

- Does not delete notes
- Does not rewrite Knowledge Core
- Work notes are stored separately from knowledge.json
