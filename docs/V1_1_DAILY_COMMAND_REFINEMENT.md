# Project NEXUS v1.1 日常コマンド強化

## 目的

v1.1では、毎日使うための入口を整理する。

v1.0では「実用版NEXUSの土台」が完成した。
v1.1では、起動直後に何をすればいいかを迷わないようにする。

## 追加する考え方

日常利用では、ユーザーが毎回細かいコマンドを覚える必要はない。

まず必要なのは次の3つ。

1. 今どこにいるか
2. システムは安全か
3. 次に何をするべきか

## 推奨コマンド

将来的に追加する候補：

- 今日のNEXUS
- 今日の作業確認
- 次にやること
- ダッシュボード案内
- 保存前チェック

## v1.1完了条件

- 日常起動時の流れが明文化されている
- 次作業確認の方針がある
- ダッシュボード案内と連動できる
