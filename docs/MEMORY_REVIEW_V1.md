# Memory Review v1

Adds read-only memory review commands for Project NEXUS v0.6.

## Commands

- 記憶レビュー
- 古い記憶候補
- 重複記憶候補
- 記憶安全確認

## Scope

Reviews:

- Knowledge Core
- Work Notes
- Project Memory
- Archive / Auto Recall settings

## Safety

- Read-only
- No deletion
- No automatic rewrite
- Candidate display only
