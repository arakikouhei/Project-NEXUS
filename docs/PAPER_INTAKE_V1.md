# Paper Intake v1

Safe arXiv metadata intake.

## Commands

- 論文ヘルプ
- 論文検索: artificial intelligence
- 論文検索: computer graphics
- 論文保存: arXiv ID
- 論文一覧
- 論文詳細: papers-xxxxxxxx

## Scope

v1 uses arXiv API and stores metadata + abstract only.

## Storage

Knowledge Core / papers

## Limits

- No bulk PDF storage
- No full-text redistribution
- Abstract-based search and storage only
