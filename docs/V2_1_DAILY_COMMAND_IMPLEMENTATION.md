# Project NEXUS v2.1 今日のNEXUS コマンド実装

## 目的

v2.1では、NEXUSを毎日起動したときに最初に見る入口を作る。

## 実装対象

- scripts/nexus_today.py
- nexus/daily/workflow.py

## 今日のNEXUSで表示するもの

1. 現在地
2. 推奨される次の段階
3. システム状態
4. 重要コマンド
5. ダッシュボード起動方法
6. 保存前チェック
7. 次にやること

## 方針

NEXUS本体を壊さないため、まずは独立スクリプトとして実装する。

将来的にコンソールコマンド `今日のNEXUS` として統合する。
