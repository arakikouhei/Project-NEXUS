# File Index v1

File Index v1 adds read-only project file listing commands for Project NEXUS v0.7.

## Commands

- ファイルインデックス
- 重要ファイル一覧
- docs一覧
- data一覧
- tools一覧
- scripts一覧
- prompts一覧

## Purpose

- Show important project directories
- List important files
- Prepare for future dashboard file browser

## Safety

- Read-only
- Does not edit files
- Does not delete files
- Ignores backup and cache directories
